Objectifs du module

Objectifs de formation

A l’issue du module, les apprenants auront des serieuses bases en dévelopement web, design, gestion de projet et création de chatbot.

  • Travailler comme les startups de façon agile
  • Déployer un site en production
  • Publier un chatbot messenger, slack, telegram…

Objectifs pédagogiques

Au cours du module, les participants apprendront à :

  • Travailler en équipe grâce à git et GitHub
  • Architecturer le design de grosses application web
  • Contruire une interface de site web responsive
  • Gérer un projet de façon agile
  • Utiliser Node.js et npm

Moyens pédagogiques

Matériel

Ce module mobilise les moyens matériels suivants : Salle de formation équipée d'ordinateurs personnels des apprenants avec Linux ou MacOS. Il est possible d'utiliser windows mais à vos risques et périls module.

Le module sera animé sous forme de :

  • Cours
  • Exercice
  • Agile Serious Game
  • Rétrospective

Pré-requis

Ce module s’adresse aux apprenants en filière Web & Digital. Pour suivre ce module, il convient au préalable de maîtriser le

Il vous est par ailleurs conseillé, en amont de ce module, d’avoir un ordinateur de développeur avec un compte GitHub associé.

Durée

Le module est d’une durée de 35 heures organisées en 5 séances.

Programme du module

DEV

  1. Lecture Git & GitHub 102
  2. Lecture Git & GitHub 103
  3. Setup Node.js
  4. Build our own classroom website

DESIGN

  1. Lecture CSS Architecture for Large-Scale UIs
  2. Live Code Build a responsive website
  3. Advanced design
  4. Lecture Design Sprint

AGILE

  1. Lecture agile 101
  2. Live code Getting Things Done
  3. Google Certification
  4. Evernote hacks for power users

CHATBOTS

  1. Lecture Chatbot 101
  2. Live code Build your first messenger bot

DEMO

  1. Finish your product
  2. DEMO Pitch your product

Bonus

Rétrospective

Echanges sur la formation réalisée.

  • Questions / Réponses sur ce qui a été vu
  • Références, ressources et partage des supports
  • Pistes d’amélioration

Suivi et evaluation

Le module sera évalué en contrôle avec l'obtention de la Certification Google for digital active.

En évaluation finale, les apprenants présentent leur application / site web devant la classe entière.
